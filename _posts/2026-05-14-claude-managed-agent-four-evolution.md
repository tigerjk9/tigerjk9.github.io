## 드리밍: AI 에이전트의 자기 성찰 루프

**드리밍**은 에이전트가 작업 기록을 되돌아보며 스스로를 개선하는 기능이다. 마치 사람이 일기를 쓰거나 회고하는 것처럼, 에이전트는 과거 세션의 메모리 저장소를 훑어보며 반복된 실수와 성공적인 워크플로를 분석한다. 이를 통해 에이전트는 쓸모 있는 기억만 선명하게 정리하고 다음 세션에 반영한다.

드리밍의 효과는 다음과 같이 요약된다.

| 구분 | 내용 | 효과 |
|---|---|---|
| 대상 작업 | 팀이 반복하는 작업, 장기 프로젝트, 여러 에이전트가 팀 취향을 공유해야 하는 경우 | 작업 효율성 향상, 일관성 유지 |
| 작동 방식 | 과거 세션 분석, 실수 패턴 및 성공 워크플로 식별, 유용한 정보 선별 및 저장 | 에이전트의 장기적인 학습 및 개선 |
| 핵심 | 작업 중 학습 내용을 넘어 세션 간 학습 구조 구축 | 지속적인 성능 향상 |

작업 중 배우는 것이 메모리라면, 드리밍은 세션 사이에 배우는 구조다.

## 아웃컴: 결과물의 품질 기준 제시

**아웃컴**은 에이전트에게 '잘했다'의 기준을 명확히 제시하는 기능이다. 단순히 "잘 만들어 줘"라고 지시하는 대신, 좋은 결과가 무엇인지 루브릭 형태로 정의한다. 에이전트는 산출물을 만들고, 별도의 채점 모델이 루브릭에 따라 결과를 평가한다.

아웃컴의 핵심은 작업자와 검수자를 분리했다는 점이다. 작업한 에이전트가 자기 합리화를 하는 대신, 독립적인 채점기가 결과만 보고 객관적으로 판단한다. 부족한 부분은 피드백하고, 에이전트는 이를 바탕으로 결과물을 개선한다.

아웃컴은 다음과 같은 작업에서 특히 효과적이다.

| 작업 유형 | 예시 | 효과 |
|---|---|---|
| 기준이 명확한 작업 | 보고서 작성, 문서 작성, 브랜드 톤에 맞는 글쓰기, 디자인 가이드 준수 | 결과물의 품질 향상, 일관성 유지 |
| 요구 사항을 빠뜨리면 안 되는 작업 | 보고서 작성 | 누락 방지 |
| 형식이 중요한 작업 | 문서 작성 | 형식 오류 감소 |

좋은 결과를 말로 정의할 수 있을수록 아웃컴의 효과는 커진다.

## 멀티에이전트 오케스트레이션: 분업과 협업의 시너지

**멀티에이전트 오케스트레이션**은 하나의 에이전트가 모든 것을 혼자 하는 대신, 리드 에이전트가 작업을 분할하고 전문 서버 에이전트에게 나눠주는 구조다. 각 서버 에이전트는 자신만의 모델, 프롬프트, 도구를 사용하여 병렬적으로 작업을 수행한다.

예를 들어 장애 원인을 찾는 경우, 각 에이전트는 다음과 같이 역할을 분담할 수 있다.

| 에이전트 | 역할 |
|---|---|
| 에이전트 A | 배포 이력 분석 |
| 에이전트 B | 에러 로그 분석 |
| 에이전트 C | 매트릭 분석 |
| 에이전트 D | 고객 문의 분석 |
| 리드 에이전트 | 각 에이전트의 결과를 종합하여 최종 판단 |

이러한 분업 구조는 작업이 복잡하고 광범위할 때 특히 유용하다. 여러 자료 출처를 분석하거나, 리서치, 분석, 문서화가 동시에 필요한 작업에 적합하다. 작은 질문에는 과도할 수 있지만, 큰 작업에서는 병렬 처리 방식이 효율성을 높인다.

## 웹훅: 작업 흐름의 자동 연결

**웹훅**은 에이전트 작업이 완료되었을 때 외부 시스템으로 신호를 보내는 연결 고리다. 에이전트 실행 후 결과를 기다릴 필요 없이, 작업 시작, 대기, 완료, 문제 발생 등의 상태 변화를 실시간으로 알 수 있다.

웹훅을 통해 에이전트 작업은 다른 업무 시스템과 자연스럽게 연결된다. 예를 들어 문서 생성이 완료되면 검수로 보내고, 분석이 완료되면 슬랙으로 알리고, 렌더링이 완료되면 품질 검사를 실행하는 등 자동화된 워크플로를 구축할 수 있다. 웹훅은 짧은 채팅에는 불필요하지만, 운영 자동화를 구축할 때는 핵심적인 역할을 수행한다.

## 요리과학·발효학의 시선으로 보면

발효는 단일 균주가 아닌 복합 미생물 생태계의 협업으로 이루어진다. 각 미생물은 특정 역할을 수행하며, 이들의 상호작용이 최종 결과물의 풍미를 결정한다. 클로드 매니지드 에이전트의 멀티에이전트 오케스트레이션 또한 이와 유사하다. 리드 에이전트는 발효조의 환경을 조성하고, 각 전문 에이전트는 특정 발효 과정을 담당하며, 이들의 협업이 고품질 결과물을 만들어낸다. 중요한 것은 각 단계에 맞는 역할을 분담하고, 전체 시스템의 균형을 유지하는 것이다.
