클로드 코드, 쓰면 쓸수록 토큰이 녹는다고? 걱정은 끝이다. **앤스로픽** 공식 플러그인 4가지로 토큰 절약과 **하네스 엔지니어링**을 동시에 잡는 방법을 소개한다. 실무 적용 데모까지 한 번에 확인한다.

## 토큰 절약, 어디서 시작해야 할까

**클로드 코드**를 사용하면서 토큰이 빠르게 소모되는 문제를 해결하기 위해, 앤스로픽 공식 마켓플레이스의 플러그인을 활용하는 방법이 제시된다. 이 영상에서는 토큰 절약에 효과적인 네 가지 플러그인을 소개하고, 실제 업무 환경에서 어떻게 적용할 수 있는지 데모를 통해 자세히 설명한다. 토큰이 무엇인지, 왜 소모되는지에 대한 기본적인 이해가 부족하다면, 짐코딩의 이전 영상을 먼저 시청하는 것이 좋다.

## 앤스로픽 플러그인 마켓플레이스 활용법

앤스로픽 공식 마켓플레이스에는 171개의 다양한 플러그인이 존재한다. 이 중에서 토큰 절약에 직접적으로 도움을 주는 플러그인은 다음과 같다.

1. **session-reports**: 토큰 사용량을 시각적으로 분석
2. **claudemd-management**: **CLAUDE.md** 파일을 효율적으로 관리
3. **serena-mcp**: 코드 검색 시 토큰 소비를 최소화
4. **hookify**: 클로드의 실수를 사전에 방지

이러한 플러그인을 설치하는 방법은 간단하다. 클로드 코드에서 `/plugins` 명령어를 입력하여 마켓플레이스에 접속한 후, 원하는 플러그인을 검색하여 설치하면 된다. 설치 후에는 `/reload-plugins` 명령어를 통해 플러그인을 활성화한다.

## session-reports: 토큰 사용량 시각화

**session-reports** 플러그인은 클로드 코드의 토큰 사용 내역을 명세서 형태로 제공한다. 프로젝트별, 스킬별, 프롬프트별로 토큰 사용량을 항목별로 보여준다. 토큰 사용량이 과도하게 발생했을 때 원인을 파악하거나, 클로드 코드의 캐시 기능이 제대로 작동하는지 확인하는 데 유용하다. **프롬프트 캐싱**은 반복되는 부분의 토큰 비용을 절감하는 핵심 기능이지만, 캐시가 깨지면 토큰 비용이 증가한다. session-reports는 이러한 캐시 동작을 점검하는 데 활용된다.

session-reports는 자연어 또는 슬래시 명령어로 호출할 수 있다. 기간 옵션을 설정하여 특정 기간 동안의 리포트를 생성할 수도 있다. 생성된 리포트는 HTML 파일 형태로 제공되며, 분석 기간, 입력/출력 토큰 비율, 캐시 처리 비율, 토큰을 많이 사용한 프로젝트 등의 정보를 확인할 수 있다. 특히, 가장 비싼 프롬프트 섹션에서는 각 프롬프트의 작업 사이클과 토큰 사용 비율을 상세하게 분석할 수 있다.

## claudemd-management: CLAUDE.md 파일 관리

**CLAUDE.md** 파일은 프로젝트에서 클로드 코드가 따라야 할 규칙을 정의하는 파일이다. 프로젝트 진행에 따라 파일 크기가 커지기 쉬운데, **claudemd-management** 플러그인은 이러한 파일을 효율적으로 관리하는 데 도움을 준다. 이 플러그인은 두 가지 주요 도구를 제공한다.

1. **revise-claudemd**: 대화가 끝난 후 새로운 명령어나 작업 패턴을 추출하여 CLAUDE.md 파일에 추가할 후보를 제시한다. 회의록 작성과 유사한 역할을 한다.
2. **claudemd-improver**: CLAUDE.md 파일을 여섯 가지 기준으로 평가하여 등급을 매기고, 작동하지 않는 명령어, 누락된 의존성, 환경 설정 등의 이슈를 파악한다. 주기적인 파일 검사에 유용하다.

## serena-mcp: 의미 기반 코드 검색

기존의 텍스트 기반 검색 방식은 불필요한 탐색을 유발하고 토큰을 낭비한다. **serena-mcp**는 코드의 의미를 이해하고, 요청에 맞는 코드를 빠르게 탐색할 수 있도록 돕는다. 코드 편집기의 "Go to definition" 기능과 유사하게, 코드의 의미를 기반으로 검색하기 때문에 토큰 절약 효과가 크다. 프로젝트 분석, 코드 사용 위치 파악 등에 유용하며, 리팩토링 작업 시에도 효율성을 높일 수 있다.

serena-mcp를 설치하려면 **UV**라는 파이썬 패키지 매니저가 필요하다. UV 설치 후, 클로드 코드에서 serena 플러그인을 설치하고 MCP를 연결하면 된다. serena-mcp가 활성화되면 대시보드를 통해 활성 도구 및 사용 내역을 확인할 수 있다.

## hookify: AI 실수를 사전에 차단

**hookify**는 **하네스 엔지니어링**의 핵심 도구로서, 클로드의 실수를 사전에 방지하여 토큰 낭비를 줄인다. **미첼 해시모토**는 하네스 엔지니어링을 "AI 에이전트가 실수를 할 때마다 그 실수가 다시 발생하지 않도록 환경을 엔지니어링하는 것"이라고 정의했다. hookify는 클로드가 자주 하는 실수를 등록하고, 해당 실수를 방지하는 규칙을 설정하여 토큰을 절약하고 작업 효율성을 높인다.

hookify 플러그인에는 다양한 스킬과 명령어가 존재한다. 실수를 등록하는 명령어, 최근 짜증났던 패턴을 추출하는 명령어, 등록된 규칙을 확인하는 명령어 등이 있다. 특히, 같은 실수를 세 번 반복했을 경우, "최근에 짜증났던 패턴 추출" 명령어를 사용하여 해당 실수를 방지하는 규칙을 추가할 수 있다.

## 데이터 과학의 이상치 탐지 관점에서 보면

데이터 과학에서 이상치 탐지(Anomaly Detection)는 정상적인 데이터 패턴에서 벗어난 특이한 데이터를 식별하는 기술이다. hookify는 클로드 코드의 비정상적인 동작, 즉 실수를 사전에 감지하고 차단하는 점에서 이상치 탐지 시스템과 유사한 역할을 수행한다. 클로드 코드 사용 과정에서 발생하는 오류 패턴을 학습하고, 이러한 패턴이 재발하지 않도록 규칙을 설정하는 것은, 데이터 과학에서 이상치를 탐지하고 제거하여 모델의 성능을 향상시키는 과정과 유사하다. 이상치 탐지 기술은 금융 사기 탐지, 제조 결함 감지, 네트워크 침입 탐지 등 다양한 분야에서 활용되며, AI 에이전트의 신뢰성과 효율성을 높이는 데에도 기여할 수 있음을 시사한다.
