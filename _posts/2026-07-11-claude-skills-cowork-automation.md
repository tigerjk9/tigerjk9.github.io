## 클로드의 차별점

클로드는 다른 AI 모델들과 비교하여 몇 가지 핵심적인 차별점을 지닌다. 이는 교사들이 업무에 클로드를 적극 활용하는 이유가 된다.

| 특징     | 클로드                                     | 챗GPT/제미나이                                     |
|--------|------------------------------------------|----------------------------------------------------|
| **글쓰기** | 양질의 서적 학습으로 사람처럼 자연스러운 글쓰기 | 인터넷의 다양한 자료 학습으로 잡다한 정보 포함 가능 |
| **코딩**   | 개발자가 선호하는 높은 코딩 능력               | -                                                |
| **에이전트** | **코워크** 기능으로 에이전트 모델 가장 잘 구현  | **코덱스**, **스파크** 등 에이전트 모델 개발 중 |
| **윤리성** | Anthropic CEO의 신뢰성, 윤리적 AI 지향       | -                                                |

클로드는 처음부터 양서 위주로 학습하여 글을 매우 잘 쓴다. 사람이 쓴 것처럼 자연스러운 글쓰기 능력은 교과 세특 등 문서를 작성할 때 큰 강점으로 작용한다. 또한 개발자들이 사랑하는 AI로 꼽힐 만큼 코딩 능력도 뛰어나다.

현재 AI는 챗봇 시대를 지나 에이전트 시대로 진입하고 있다. 클로드 코워크는 목표를 주면 직접 일을 처리하고 결과물까지 만들어내는 **에이전트** 기능이 가장 잘 구현된 사례다. 더불어 클로드를 만든 Anthropic의 경영진은 윤리적인 AI 개발에 힘쓰며, 실제 미국 국방부와의 계약 사례에서 사용자의 개인 정보 보호와 최종 판단의 인간 역할 강조로 신뢰성을 입증했다.

다만, 클로드의 고급 기능은 유료 구독(최소 20달러)을 요구하며, 작업을 수행할 때 토큰을 많이 소비한다.

## 반복 업무 자동화, 클로드 스킬

**클로드 스킬**은 클로드가 작업을 시작하기 전에 읽는 상세한 업무 매뉴얼이다. 이는 교사의 반복적인 업무를 자동화하는 데 매우 효과적이다. 기존 **챗GPT**의 나만의 GPT나 **제미나이**의 젬스 기능과 비교하여 몇 가지 강력한 장점을 지닌다.

| 스킬의 장점        | 상세 설명                                                                                                                               |
|------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| **반복 설명 감소** | 긴 설명 없이 한두 마디 명령으로 충분하다. AI가 매뉴얼을 읽고 작업을 수행한다.                                                           |
| **학교 양식 유지** | 학교 가정통신문, 활동지, PPT, 엑셀 등 고유 양식을 학습시켜 내용만 입력하면 해당 양식으로 결과물을 생성한다. 100%에 가까운 양식 구현율을 보인다. |
| **품질 동일**      | 누가 사용하든, 언제 사용하든 항상 동일한 품질의 결과물을 도출한다.                                                                      |
| **공유 용이성**    | 스킬 파일을 다른 교사와 공유하여 동일한 업무 자동화 환경을 구축한다.                                                                    |

**클로드 스킬** 생성은 간단하다. 채팅창에 "이런 스킬 만들어 줘"라고 요청하면 클로드가 필요한 정보를 질문하고, 답변을 토대로 스킬을 만든다. 대화 중 "스킬을 수정해 줘"라고 말하면 기존 스킬을 즉시 수정할 수 있다. 생성된 스킬은 사용자 지정 메뉴에서 다운로드하거나 다른 사람이 만든 스킬을 업로드하여 활용한다.

예를 들어, 공문 파일을 업로드하고 "공문을 메시지로"라고 명령하면, 클로드는 학습된 스킬 매뉴얼에 따라 간결한 업무 메신저용 메시지를 자동으로 생성한다. 학교 가정통신문 양식을 학습시키면 내용 입력만으로 양식에 맞는 통신문을 만든다. PPT 양식을 학습시키면 깔끔한 수업용 PPT 자료도 제작한다. 한글(HWP) 파일 양식의 경우, 별도의 HWP 변환 스킬을 먼저 업로드하여 활용하면 더욱 편리하다.

## 클로드 코워크, 에이전트 AI의 시대

클로드 코워크는 단순한 챗봇이 아니라 에이전트 AI다. 질문에 답하는 데 그치지 않고 목표를 받으면 직접 작업을 계획하고 실행하며, 결과물까지 생성하여 사용자에게 제공한다.

| 분류     | 챗봇                                   | 에이전트 (클로드 코워크)                         |
|--------|--------------------------------------|-----------------------------------------------|
| **역할** | 질문에 답하는 동료                     | 내 PC에서 직접 일을 처리하는 대리인             |
| **방식** | 질문-답변                              | 목표-계획-실행-판단(재계획/재실행)-결과물       |
| **토큰** | 적게 사용                              | 많이 사용 (복잡한 작업 수행)                  |

**클로드 채팅**과 클로드 코워크는 활용 방식에서도 차이를 보인다.

| 구분           | 클로드 채팅                                | 클로드 코워크                                            |
|--------------|------------------------------------------|--------------------------------------------------------|
| **사용 환경**  | 웹 브라우저, 스마트폰 앱                     | **클로드 데스크탑 앱** (PC 설치 필수)                  |
| **파일 처리**  | 한 번에 최대 20개, 파일당 30MB            | 폴더 내 파일 수 제한 없음 (폴더 내 모든 파일 처리 가능) |
| **결과물 저장** | 채팅창 내 텍스트 답변 또는 다운로드 파일       | 내 폴더에 실제 파일로 직접 저장                          |
| **작업 유형**  | 번역, 요약, 질의응답 등 단발성 작업           | 파일 대량 정리, 반복 작업 등 여러 단계의 복잡한 작업 자동 처리 |
| **고급 기능**  | 웹 검색으로 자료 조사 (텍스트 답변)       | 웹 검색, 자료 조사, DOCX 파일 저장, 노션 업로드, 카톡 전송 등 복합 작업 |

## 코워크 시작을 위한 3단계

클로드 코워크를 효과적으로 사용하려면 세 가지 사전 준비가 필요하다.

1.  **클로드 데스크탑 앱 설치**:
    *   **코워크**는 사용자의 컴퓨터에서 직접 작업을 수행하므로, 반드시 **클로드 데스크탑 앱**을 설치해야 한다.
    *   이 앱은 AI에게 컴퓨터 접근 권한을 부여하지만, 안전을 위해 지정된 특정 폴더 내에서만 작업이 이루어지도록 제한한다.
    *   앱 설치는 클로드 웹사이트나 확장 프로그램 메뉴에서 가능하다.

2.  **클로드 포 크롬 설치**:
    *   이 크롬 확장 프로그램은 클로드에게 인터넷 검색 및 특정 웹사이트에 대한 접근 권한을 부여한다.
    *   일반 웹 검색 기능으로는 접근하기 어려운 로그인된 사이트나 유튜브 채널 내용도 클로드 포 크롬으로 분석할 수 있다.
    *   현재 데모 버전으로 토큰 소모가 많아 아직은 일반 웹 검색 기능으로도 충분히 활용할 수 있다.

3.  **커넥터 연결**:
    *   **커넥터**는 클로드와 **캠바(Canva)**, **노션(Notion)**, **구글 드라이브(Google Drive)**, **지메일(Gmail)** 등 다양한 외부 앱을 연결하는 기능이다.
    *   이 기능으로 클로드는 외부 앱의 데이터를 가져오거나, 작업 결과를 해당 앱에 저장할 수 있다.
    *   특히 **플레이 MCP(Play MCP)**를 연결하면 카카오톡, 네이버 등 국내 서비스를 클로드와 손쉽게 연동하여 활용 범위가 크게 확장된다.

## 교사 업무 전반의 코워크 활용 사례

클로드 코워크는 교사의 광범위한 업무 영역에서 실질적인 자동화를 구현한다.

*   **행정 업무 자동화**:
    *   수신 공문을 접수 일자 기준으로 폴더별로 자동 정리한다.
    *   다운로드 폴더 내의 문서, 비디오, 설치 파일 등을 유형별로 분류하여 정리한다.
    *   현장 체험학습 영수증을 모아 엑셀 파일로 정산 보고서를 자동으로 생성한다.
    *   AI 디지털 선도학교 운영 계획서 공문과 양식을 바탕으로 운영 계획서 초안을 작성한다.

*   **평가 업무 지원**:
    *   학생 탐구 보고서 등 수행평가를 평가 기준에 따라 채점하고, 엑셀 파일로 정리된 채점 코멘트까지 생성한다. AI 평가 결과는 교사의 평가와 비교하며 놓칠 수 있는 부분을 발견하는 데 도움을 준다.
    *   서울대학교 합격생들의 세특(세부능력 특기사항)을 분석하여 공통 패턴과 작성 실전 팁이 담긴 보고서를 DOCX 파일로 생성한다.

*   **수업 업무 혁신**:
    *   최소 성취 수준 보장지도 활동지를 자동으로 제작한다.
    *   매주 특정 요일과 시간에 최신 로봇 뉴스를 검색하여 PPTX 형식의 브리핑 자료와 DOCX 형식의 배경 및 시사점 정리 파일을 구글 드라이브에 저장한다.
    *   수업에 활용할 인물 소개 PPT를 스토리 형식으로 조사하고 제작한다.
    *   폴더 내의 다양한 형식(한글, 텍스트, 엑셀, 구글 문서, PDF, PPT)의 자료를 종합하여 보고서로 만든다.

*   **개인 및 연구 업무**:
    *   중학교 3학년 자녀를 위한 고등학교 선택 가이드 정보를 유튜브와 웹 리서치로 조사해 HTML 파일 보고서로 만든다.
    *   구글 캘린더, Gmail, 노션에 있는 개인 일정과 업무 DB를 종합하여 매일 아침 HTML 파일 보고서를 만들고, 핵심 내용을 요약하여 카카오톡으로 전송한다.
    *   AI 관련 최신 업데이트 내용을 주 1회 HTML 파일로 정리하여 강의 자료 리서치를 돕는다.
    *   가족 여행 경비를 지원하는 지자체 정보를 주 1회 조사하여 알려준다.
    *   모의고사 성적 데이터를 분석하여 학생별 변화 추이, 과목별 취약점, 집중 관리 대상 학생 목록 및 상담 코멘트까지 생성한다.
    *   학생 상담 녹음 파일을 엑셀 파일에 날짜별로 누적 기록하는 상담 일지 자동화도 가능하다.

## 업무 자동화 설계 5단계

클로드 코워크로 업무를 자동화하는 과정은 5단계로 설계할 수 있다.

| 단계         | 설명                                            | 예시                                                                                 |
|------------|-------------------------------------------------|------------------------------------------------------------------------------------|
| **1. 입력**  | 자료를 가져올 원천을 지정한다.                  | 폴더, 구글 드라이브, Gmail, 노션, 웹 검색, 파일 첨부, Play MCP 앱                   |
| **2. 처리**  | AI에게 수행할 작업을 명령한다.                  | 요약, 분류, 분석, 비교, 변화 추이 파악, 보고서 작성 등                               |
| **3. 출력 형식** | 결과물을 어떤 파일 형식으로 받을지 지정한다.    | HWP, PPTX, XLSX, DOCX, HTML, 채팅 텍스트 등                                          |
| **4. 저장/전달** | 결과물을 어디에 저장하거나 전달할지 지정한다.   | 로컬 폴더, 구글 드라이브, 노션 DB, Gmail 초안, 카카오톡으로 전송                       |
| **5. 실행 트리거** | 작업이 언제 시작될지 조건을 설정한다.           | 폴더 내 파일 감지, 매주 특정 요일 및 시간, 즉시 실행, 내가 명령할 때만 실행           |

이 다섯 단계를 조합하면 매우 복잡하고 다양한 형태의 업무 자동화를 구현한다. 클로드 코워크는 이 모든 과정을 한 번에 처리할 수 있어 교사들이 교육의 본질에 집중하고 행복한 교육 환경을 만드는 데 크게 기여한다.

## 인지과학의 시선으로 보면

클로드 코워크와 같은 AI 에이전트의 활용은 인지과학 분야의 **분산 인지(Distributed Cognition)** 개념과 연결된다. 이는 인간의 인지 과정이 개인의 두뇌에만 한정되지 않고, 환경 속의 도구, 기술, 다른 사람 등 다양한 요소들과 상호작용하며 확장된다는 관점이다. 클로드 코워크는 교사의 외부에 존재하는 강력한 인지 도구로 기능하며, 교사가 수행하던 복잡한 정보 처리, 계획, 실행, 평가 과정을 AI에 위임함으로써 인지 부하를 줄이고, 교사는 더 고차원적인 판단과 창의적인 교육 활동에 집중한다. 이는 마치 인간이 계산기나 필기도구를 사용하여 인지 능력을 확장하는 것과 같은 방식으로, AI가 교사의 업무 환경에서 '인지적 동반자' 역할을 하는 진화된 형태의 분산 인지를 보여준다.
