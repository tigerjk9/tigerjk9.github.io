디자인 교육 기업 Design+Code의 설립자 멩 토(Meng To)가 AI 에이전트를 위한 스킬 라이브러리를 공개했다. 이 라이브러리는 디자이너와 빌더가 **Codex**, **Claude**, **Cursor** 등 다양한 AI 코딩 에이전트를 활용해 사용자 인터페이스, 프런트엔드 시스템, 자동화 워크플로를 효율적으로 구축하도록 돕는다. 좋은 프롬프트를 재사용 가능한 자산으로 전환하는 방법을 제시한다.

## Meng To의 에이전트 스킬이란?

이 라이브러리는 **에이전트 스킬**의 엄선된 컬렉션이다. 풍부한 사용자 인터페이스, 프런트엔드 시스템, 에이전트 루프, 자동화 및 재사용 가능한 워크플로를 구축하는 데 활용된다. **Codex**, **Claude Design**, **Aura Build**, **Lovable** 등 여러 에이전트 스택에서 참조 자료를 프롬프트로 바꾸고, 상세한 랜딩 페이지를 생성하며, 재사용 가능한 프런트엔드 워크플로를 구축한다.

각 스킬은 `SKILL.md` 파일 형태의 간결한 플레이북이다. 여기에는 선택적 참조 자료, 아티클, 스크립트, 에셋이 포함된다. 목표는 명확하다. 좋은 프롬프트, 워크플로, 스타일 시스템, 캡처 레시피, 디버깅 습관을 에이전트가 로드하고 따를 수 있는 버전 관리 파일로 만드는 것이다. 이 스킬들은 기본적으로 휴대성을 지닌다. 사용자가 로컬 에이전트 명령을 통해 프로젝트별 컨텍스트를 제공하지 않는 한, 모든 사용자, 저장소, 작업 공간에서 작동한다.

스킬은 다음과 같은 상황에서 유용하다.
*   반복 가능한 디자인 방향을 원할 때
*   절차적 구현 단계를 정의할 때
*   복사/붙여넣기 가능한 코드 조각이 필요할 때
*   일반적인 함정과 안전장치를 마련할 때
*   일회성 채팅 답변 대신 재사용 가능한 워크플로를 구축할 때

## 주요 웹 디자인 워크플로 살펴보기

라이브러리에는 웹 디자인에 특화된 대표 워크플로가 포함된다.

| 스킬 명칭                 | 설명                                                                                                | 핵심 기능                                                                                                  |
|:------------------------|:----------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------|
| **Video to Super Prompt** | 디자인, 랜딩 페이지 또는 애니메이션의 화면 녹화 영상을 **Fable 5**가 HTML을 한 번에 생성하도록 돕는 매우 상세한 프롬프트로 변환한다. | 녹화 영상 분석, 상세 프롬프트 생성, HTML 원샷 빌드                                                      |
| **HTML to Interaction Prompts** | 기존 HTML 페이지(예: Aura Build 페이지)를 하나의 섹션, 애니메이션, 버튼, 호버 상태 또는 WebGL 효과를 위한 재사용 가능한 프롬프트로 전환한다. | 기존 HTML 분석, 상호작용 프롬프트 추출, 재사용성                                                            |
| **Stitched Full Page Capture** | 랜딩 페이지의 특정 영역(히어로 섹션 등)이 아닌 전체 페이지를 캡처한다. 에이전트에게 구조, 페이싱, 시각적 계층에 대한 완전한 참조 자료를 제공한다. | 전체 페이지 캡처, 구조 및 계층 정보 제공, 정확한 참조                                                       |
| **Daily UI Inspiration** | 브라우징, 캡처, 참조 연구, 프롬프트 생성을 유용한 에이전트 루프와 결합한다. 인상적인 랜딩 페이지를 상세한 프롬프트 팩으로 변환한다. | UI 영감 수집, 자동화된 프롬프트 생성, 상세 프롬프트 팩 제공                                                   |

이 워크플로는 AI 에이전트가 디자인 작업의 초기 단계부터 구현에 이르기까지 일관되고 효율적인 지원을 제공하도록 설계된다.

## 에이전트 스킬의 핵심 철학

스킬 라이브러리는 네 가지 주요 철학을 기반으로 한다.

*   **프롬프트는 자산이다**: 한 번 효과적인 프롬프트는 재사용되어야 한다. 프롬프트를 파일로 저장하고, 버전 관리하며, 라이브러리와 스타일카드를 구축해야 한다.
*   **명세가 분위기를 이긴다**: 일관된 결과물을 가장 빠르게 얻는 방법은 명확한 제약 조건, 명확한 계층 구조, 그리고 '1~2가지만 변경하는' 반복 방식이다.
*   **참고 자료가 긴 설명을 이긴다**: 스크린샷과 예시는 글꼴, 간격, 색상, 레이아웃 리듬, 아이콘 스타일 등 많은 정보를 전달한다.
*   **스킬은 운영 절차이다**: 좋은 스킬은 에이전트에게 언제 스킬을 사용해야 하는지, 무엇을 먼저 해야 하는지, 어떤 기본값을 적용해야 하는지, 어떤 실수를 피해야 하는지 정확히 알려준다.

## 다양한 에이전트 지원 및 구조

이 스킬 라이브러리는 의도적으로 평이한 마크다운과 폴더 기반 형식을 사용한다.

*   **Codex**: 관련 `SKILL.md` 파일을 로드한 후 작업을 수행한다.
*   **Claude Code**: `CLAUDE.md`에서 관련 스킬을 참조하거나, Claude 스킬 설정으로 복사하거나, `SKILL.md`를 직접 작업 컨텍스트로 연다.
*   **Cursor**: Cursor 규칙 또는 채팅 컨텍스트를 특정 스킬 폴더로 지정한다. 재사용 가능한 코드 조각, 제약 조건, 기본값을 쉽게 붙여넣기 할 수 있도록 유지한다.
*   **다른 에이전트**: 가장 적합한 스킬을 먼저 읽고, 그 단계와 연결된 참조 자료를 따른다.

저장소 구조는 `agent-skills/<category>/<skill-name>/` 형식이다. 각 스킬 폴더에는 필수 `SKILL.md` 파일과 선택적으로 `REFERENCES.md` (링크만 포함), `ARTICLE.md` (장문 설명), `assets/` (이미지, 템플릿), `scripts/` (헬퍼 스크립트)가 포함된다. `SKILL.md`는 에이전트가 로드하고 따르는 스킬 내용을 담는다. 스킬은 절차적(단계, 패턴, 안전장치)이어야 하며 백과사전식 설명을 피한다. 명시적인 트리거("다음과 같은 상황에서 사용한다")와 기본값(기간, 간격, 계층, 명령어, 승인 확인)을 선호한다.

## 스킬 라이브러리 활용 예시

이 라이브러리에는 현재 **75개**의 스킬이 네 가지 주요 카테고리에 걸쳐 제공된다.

*   **Codex 워크플로 (10개)**: 작업 감사 및 검증, 고객 이메일 초안 작성, UI 영감 캡처, HTML을 상호작용 프롬프트로 전환, 웹 애니메이션 최적화 등 반복 가능한 Codex 작업 스킬을 포함한다.
*   **미디어 (2개)**: **Aura Assets**나 **Unsplash**에서 디자인 및 마케팅 이미지를 소싱하는 스킬이 있다.
*   **UI (1개)**: `design-first-ui-prompting` 스킬은 목표, 형식, 레이아웃, 유형, 색상, 제약 조건을 포함하는 프롬프트 템플릿과 2단계 타이포그래피 워크플로 등 디자인 우선 UI 프롬프트 시스템을 제공한다.
*   **웹 디자인 (62개)**: 랜딩 페이지, 가격 페이지, Tailwind CSS, GSAP, Three.js, WebGL, Matter.js, CSS 경계 그라데이션, 프로그레시브 블러 등 웹 개발 및 UI 구현을 위한 광범위한 스킬을 포함한다. 모션, 스크롤, WebGL, 캔버스 및 3D, CSS 처리, 레이아웃 시스템, 시각 스타일 및 페이지 분위기 등 다양한 하위 범주를 포괄한다.

## 소프트웨어 공학의 시선으로 보면

이 에이전트 스킬 라이브러리는 소프트웨어 공학에서 강조하는 **모듈화**와 **재사용성** 원칙을 AI 에이전트의 작업 흐름에 적용한 사례이다. 복잡한 디자인 및 개발 작업을 개별 스킬이라는 작은 모듈로 분리하고, 각 스킬을 특정 상황에서 재사용하도록 명세화한다. 이는 코드 베이스의 재사용성을 높여 개발 효율성을 증진하고 유지보수를 용이하게 하는 소프트웨어 공학의 오랜 목표를 AI 프롬프트 엔지니어링 영역으로 확장한다.
