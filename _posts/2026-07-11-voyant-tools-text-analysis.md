## 보얀트 툴즈는 무엇인가

보얀트 툴즈는 웹 기반의 **오픈소스 텍스트 분석 환경**이다. 스테판 싱클레어(Stéfan Sinclair)와 제프리 록웰(Geoffrey Rockwell)이 개발을 주도했다. 이 도구는 특히 **디지털 인문학** 분야에서 광범위하게 사용된다. 복잡한 프로그래밍 지식이나 별도의 소프트웨어 설치 없이 웹 브라우저 접속만으로 방대한 텍스트 데이터를 여러 관점에서 분석하고 시각화한다.

## 코딩 없이 텍스트를 분석하는 방법

보얀트 툴즈는 사용자가 분석하고자 하는 텍스트를 여러 방식으로 쉽게 입력하도록 지원한다. 다음 세 가지 방법으로 텍스트를 업로드할 수 있다.

*   **직접 붙여넣기**: 분석할 텍스트를 복사하여 도구에 바로 붙여넣는다.
*   **파일 업로드**: PDF, Word, TXT 파일 등 다양한 형식의 문서 파일을 직접 업로드한다.
*   **URL 입력**: 분석하고 싶은 웹페이지의 URL을 입력하면 해당 페이지의 텍스트를 가져온다.

이렇게 입력된 텍스트는 보얀트 툴즈의 다양한 분석 및 시각화 도구로 처리된다.

## 텍스트를 시각화하는 주요 기능

보얀트 툴즈는 입력된 텍스트를 분석하여 다섯 가지 주요 패널로 다양한 시각화 결과를 제공한다. 각 패널은 텍스트의 특정 측면을 직관적으로 보여준다.

| 패널 이름    | 설명                                                                                                                              |
| :----------- | :-------------------------------------------------------------------------------------------------------------------------------- |
| **Cirrus**   | 문서에서 가장 자주 등장하는 단어들을 추출하여 빈도수에 비례하여 글자 크기를 키운다. 워드 클라우드 형태로 핵심 키워드를 시각화한다. |
| **Trends**   | 특정 단어가 문서 전체나 여러 문서에 걸쳐 얼마나 자주 등장하는지, 그 분포와 변화 흐름을 꺾은선 그래프로 보여준다.                   |
| **Contexts** | 특정 단어를 검색하면 그 단어가 사용된 앞뒤 문맥을 표 형태로 나열한다. 단어의 실제 의미와 용례를 파악할 때 유용하다.                |
| **Reader**   | 분석 중인 원문 텍스트를 보여주는 창이다. 분석 과정에서 특정 단어가 원문의 어느 위치에 있는지 바로 확인한다.                        |
| **Bubblelines** | 특정 단어들이 문서 전체의 어느 위치에서 어느 정도의 빈도로 나타나는지를 동그란 버블 형태로 시각화한다. 문서의 맥락적 흐름을 파악하는 데 도움을 준다. |

이 패널들은 텍스트의 양적 및 질적 분석을 동시에 가능하게 하여, 복잡한 데이터를 시각적으로 이해하기 쉽게 만든다.

## 누가 보얀트 툴즈를 활용하는가

보얀트 툴즈는 다양한 분야의 사용자들에게 유용하게 쓰인다.

*   **연구자 및 학생**: 문학 작품, 역사적 연설문, 학술 논문 등을 분석하여 숨겨진 패턴이나 핵심 키워드를 찾는다.
*   **데이터 분석가 및 마케터**: 방대한 양의 고객 리뷰나 뉴스 기사 데이터에서 주요 트렌드 단어를 뽑아내는 데 활용한다.
*   **일반 사용자**: 긴 문서나 책을 읽기 전에 어떤 내용이 주를 이루는지 핵심 키워드로 빠르게 파악하는 데 도움을 받는다.

이처럼 보얀트 툴즈는 전문가부터 일반인까지 텍스트 데이터에 대한 접근성과 분석력을 높이는 데 기여한다.
