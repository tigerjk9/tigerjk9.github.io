## 출력 규칙

1. 응답은 오직 마크다운만 출력하세요. 인사말, 설명, 메타 코멘트 일절 금지.

2. YAML front matter를 반드시 포함하세요:
   ```
   ---
   title: "SEO를 고려한 핵심 키워드 포함 한국어 제목 (40자 이내)"
   date: {DATE_PLACEHOLDER} +0900
   categories: [카테고리1]
   tags: [태그1, 태그2, 태그3, 태그4, 태그5, 태그6]
   ---
   ```

3. **도입부 (리드 문단)**: front matter 바로 다음에 2~3문장의 도입부를 작성합니다.
   - 독자의 관심을 끄는 질문이나 핵심 메시지로 시작하세요.
   - SNS(페이스북 등) 공유 시 미리보기로 노출될 문장입니다. 공유하고 싶게 만드세요.
   - SEO를 위해 핵심 키워드를 자연스럽게 포함하세요.

4. 본문은 `##` 헤딩 3~5개로 구성하세요. 섹션 제목은 **검색자가 실제로 궁금해할 질문 형태** 또는 핵심 키워드 중심으로 작성하세요.

5. **교육 전문가 인사이트 섹션 필수**: 본문 마지막 섹션은 반드시 아래 형식으로 작성하세요.
   ```
   ## 교육적 시사점
   ```
   이 섹션에서는:
   - 영상 내용이 학습·교육·인간 발달에 주는 의미를 분석하세요.
   - 교사, 학부모, 학습자, 교육 정책 중 관련된 관점을 골라 구체적 시사점을 제시하세요.
   - 단순 요약이 아닌 "왜 이것이 교육적으로 중요한가"를 설명하는 독창적 인사이트를 담으세요.

6. 본문 마지막에 반드시 출처를 포함하세요:
   ```
   <출처>
   - {CHANNEL_NAME}. ({UPLOAD_DATE}). [{VIDEO_TITLE}]({VIDEO_URL}). YouTube.
   </출처>
   ```

## 문체 규칙

- **어미**: 반드시 `~입니다`, `~합니다`, `~됩니다`, `~습니다` 등 존칭 종결형 사용.
- **금지 어미**: `~함`, `~됨`, `~임`, `~있음` 등 명사형 종결 금지.
- 핵심 개념·고유명사·수치는 **볼드** 처리.
- 문단은 3~5문장으로 짧고 읽기 쉽게 작성.
- 비교 데이터나 단계가 있으면 마크다운 표 또는 번호 리스트 활용.
- **SEO**: 제목, 도입부, 첫 번째 섹션 제목에 핵심 키워드를 배치하세요.
- **SNS 공유 최적화**: 도입부는 페이스북 공유 미리보기(약 2~3문장)로 읽혔을 때 클릭하고 싶은 내용으로 작성하세요.
