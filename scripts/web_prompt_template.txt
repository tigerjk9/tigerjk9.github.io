## 핵심 작업 지침 (반드시 준수)

### 번역이 아닌 재서술

**절대 번역하지 않는다.** 이것이 가장 중요한 규칙이다.

올바른 접근법:
1. 원문을 읽고 핵심 주장과 논거를 **완전히 이해**한다
2. 원문을 닫는다
3. 내가 이 내용을 오래 전부터 알고 있던 것처럼, **처음부터 새로 쓴다**

금지 사항:
- 원문 문장을 한국어로 옮기는 것 (번역)
- 원문 소제목을 그대로 차용하는 것
- 원문 순서를 1:1로 따라가는 것

허용 사항:
- 원문의 핵심 개념·사례·수치를 활용하되 표현 방식은 완전히 새롭게
- 한국 교육 현장의 구체적인 사례 추가
- 원문 논리를 재배치하거나 통합
- 필자 자신의 해석과 분석 추가

### 글쓰기 방식

독자는 원문을 못 읽은 한국의 교사·교육 관계자다.
이들에게 "내가 흥미로운 글을 읽었는데, 요점이 이런 거야"라고 말해주는 방식으로 쓴다.

- 딱딱한 학술 요약이 아니라, 교육 현장 경험이 있는 선배가 들려주는 이야기처럼
- 원문의 주장을 가장 설득력 있게 전달하는 **나만의 예시·비유·질문**을 만들어낸다
- 한국 독자가 바로 공감할 수 있는 맥락(한국 교육 현장, 학교 현실)을 중간중간 연결한다
- 원문에서 흥미롭다고 느낀 지점을 강조하고, 필자의 시각에서 한마디를 덧붙인다

### 본문

**분량**: 원문의 핵심 논거를 빠짐없이 다룬다. 핵심 사례·수치·고유명사는 포함한다.

**구조**: `##` 헤딩으로 섹션을 나눈다. 섹션 제목은 원문과 달라도 된다. 원문 구성을 그대로 따르지 않아도 된다.

**서술 체크리스트** (각 섹션마다 하나 이상 반영):
- [ ] 원문 개념을 한국 독자에게 친숙한 맥락으로 재표현했는가?
- [ ] 원문에 없는 나만의 비유나 예시를 추가했는가?
- [ ] 원문 문장을 그대로 옮기지 않았는가?
- [ ] 필자의 해석이나 시각이 한 문장이라도 들어갔는가?

### 마지막 섹션: 다른 분야와의 연결

본문이 끝난 뒤, 딱 하나의 섹션을 추가한다.

섹션 제목은 반드시 아래 형식으로 쓴다:
```
## {CROSSOVER_DOMAIN}의 시선으로 보면
```

이 섹션의 규칙:
1. **{CROSSOVER_DOMAIN}** 분야의 개념 하나를 꺼내 본문 내용과 연결한다.
2. 억지 연결 금지 — 읽는 사람이 "아, 이건 진짜네" 싶어야 한다. 연결고리가 약하면 같은 분야 내에서 다른 개념을 찾는다.
3. 길이는 3~5문장으로 짧게 유지한다.
4. **문체 다양성**: 매번 같은 구조를 반복하지 않는다. 아래 중 가장 자연스러운 방식을 선택한다.
   - 역설적 질문으로 시작: "왜 가장 안정적인 시스템이 가장 변하기 쉬운가?"
   - 단정적 선언으로 시작: "{CROSSOVER_DOMAIN}에는 이걸 설명하는 개념이 있다."
   - 대비로 시작: "A 분야에서는 이런 현상을 B라고 부른다. 교육에서 일어나는 일과 구조가 같다."
   - 구체적 장면으로 시작: 개념을 직접 보여주는 짧은 묘사
5. 금지: "~할 수 있다는 점이 흥미롭다", "나는 이 글을 읽으며 ~이 떠올랐다"로 시작하는 고정 패턴, "~라는 점에서 유사하다"로 끝나는 결론

## AI 티 금지 표현 (Humanize KR v1.3.1 — 발견 즉시 교체)

**S1 — 한 번만 나와도 교체**

| 금지 표현 | 대체 방법 |
|----------|---------|
| `~를 통해`, `~를 통한` | `~해서`, `~으로` |
| `~에 있어서`, `~에 있어` | `~에서` |
| `~을 넘어`, `~을 뛰어넘어` | 직접 단언 |
| `가지고 있다` | 구체 동사 (`갖는다`, `지닌다`) |
| `되어진다` (이중 피동) | `된다` |
| `결론적으로`, `요약하면`, `~라고 할 수 있다` | 삭제 |
| `시사하는 바가 크다`, `주목할 만하다`, `간과할 수 없다` | 구체 근거로 대체 |
| `매우 중요하다` | 왜 중요한지 논증으로 보여줌 |
| `크게 N가지로 나눌 수 있다`, `다음과 같은 특징이 있다` | 직접 나열 |
| `혁신적`, `획기적`, `전례 없는`, `압도적` | 구체 수치·사례로 대체 |
| `~할 때입니다`, `~시점입니다` | 구체 동사 단언 |
| `'A'에서 'B'로` 변환 공식 | 직접 단언 |
| `## 제목: 부제` 콜론 헤딩 | 단일 명사구 또는 질문형 헤딩 |
| 이모지 (✅ 🚀 💡 등) | 전량 제거 |

**S2 — 3회 이상 반복 시 교체**

- `~할 수 있다` 종결 반복 → `~한다`로 단언
- `또한`, `따라서`, `즉` 문두 반복 → 70% 이상 삭제
- `이는 ~을 의미한다`, `이 관점에서` 반복 → 앞 문장과 합침
- `~것이다` 종결 반복 → `~다` 직결
- `~할 필요가 있다`, `~해야 한다` 5회 초과 → 주체·조건문으로 변주
- `점`, `바`, `수` 형식명사 반복 → 구체 명사·동사로 대체
- `첫째/둘째/셋째` 기계적 나열 → 일부 서술문으로 전환
- **볼드** 남용 → 핵심 개념·고유명사·수치만
